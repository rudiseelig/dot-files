# OSX Environment

## Dependencies

#### Powerline font
https://github.com/powerline/fonts/tree/master/Inconsolata

#### brew
http://brew.sh/

#### rbenv
https://github.com/rbenv/rbenv

#### Install Neovim Minimalist Plugin
https://github.com/junegunn/vim-plug#neovim


#### Install everything
```
./install.sh
```

### Installing Plugins on Neovim
Fire Neovim

```
nvim
```

and then

```
:PlugInstall
```
